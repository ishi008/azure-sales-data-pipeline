
# End-to-End Data Pipeline on Azure for Sales Analytics

## Overview
This project demonstrates a complete data engineering workflow using Microsoft Azure. 
The goal was to build an automated data pipeline to ingest, clean, transform and visualize sales data.

## Architecture
```
Data Source → Azure Blob Storage → Azure Data Factory → Azure Databricks (PySpark) → Azure SQL Database → Power BI Dashboard
```

## Tools Used
- Azure Blob Storage
- Azure Data Factory
- Azure Databricks (PySpark)
- Azure SQL Database
- Power BI
- Python & SQL

## Key Learnings
- Understanding of ETL workflows
- Cloud data storage & processing concepts
- Data cleaning using PySpark
- Dashboard storytelling for business insights
